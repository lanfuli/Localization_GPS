Vehicle Hardware


Elcano is an open source platform for both hardware and software. The website makes the source code and instructions for building the hardware available. The user who wishes to try the system must either build his own robot or use the Elcano control code on the simulator. This project is under construction. Instructions will be updated as hardware and software is implemented. If you build a robot to these specifications, please contact Tyler at TFolsom dot com to add it to this list.
